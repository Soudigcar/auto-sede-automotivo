# AUTO CONTROLE AUTOMOTIVO — Playbook

## Development Decision

The project must be developed in parts, with each step validated before moving to the next one.

## Language Standard

- Technical structure in English.
- Interface and operational labels in Brazilian Portuguese.

## Official MVP Stack

- GitHub Repository
- GitHub Codespaces
- Next.js
- React
- TypeScript
- Tailwind CSS
- Supabase PostgreSQL
- Supabase Auth
- Supabase Storage
- Vercel

## Current Phase

Phase 1: Database and application foundation.

## MVP Priorities

1. Authentication by role.
2. Master dashboard.
3. Store registration.
4. Prospector registration.
5. Street survey.
6. Quick registration.
7. Lead assignment to store.
8. Store lead pipeline.
9. Sale confirmation.
10. Loss registration.
11. Basic dashboard metrics.
