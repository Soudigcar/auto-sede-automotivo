# Preview Access Guide

## Current status

The project is connected to GitHub and Supabase, but it does not have a public Vercel preview URL yet.

## Repository

https://github.com/Soudigcar/auto-sede-automotivo

## Recommended preview path

1. Open the repository on GitHub.
2. Open the project in GitHub Codespaces.
3. Configure environment variables from Supabase Project Settings.
4. Run the app with npm install and npm run dev.
5. Open the forwarded port generated by Codespaces.

## Main routes to review UX

- /routes
- /login
- /master/dashboard/live
- /master/stores
- /prospector/live
- /store/live
- /store/operation
- /pre-sales

## Public link

A public link will exist only after deploying the GitHub repository to Vercel and adding the same Supabase environment variables in Vercel Project Settings.
